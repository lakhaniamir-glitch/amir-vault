## Etsy Platform Constraints (2026)

### Titles
- **Max**: 140 characters
- **First 40-50 characters are critical** — this is what shows in search results before truncation
- Front-load the primary keyword phrase (exact match ranks higher)
- Use pipe `|` separators between keyword clusters
- Never repeat the same word more than once in a title
- No ALL CAPS, no promotional language ("best seller", "limited time", "free shipping")
- No trademarked terms you don't own
- Commas and pipes signal distinct phrases to the algorithm
- Read aloud test: if it sounds robotic, rewrite it

### Tags
- **Exactly 13 tags per listing** — every empty slot is a missed search opportunity
- **Max 20 characters per tag**
- Tags are **phrase-matched** — multi-word phrases outperform single words
- Tags should **expand** keyword coverage, NOT duplicate the title
- No repeated words across tags when possible (e.g., don't use "tungsten ring" AND "tungsten band" — combine or vary)
- Etsy auto-corrects typos, so never waste tags on misspellings
- Categories and attributes already function as implicit tags — don't duplicate those either
- Exact-match tags rank higher than broad-match
- Include occasion/intent tags: "gift for him", "anniversary gift", "groom gift"
- Include synonym/alternate phrasing tags the title doesn't cover
- Seasonal tags (holiday gifting) should rotate quarterly

### Descriptions
- Etsy indexes the **first 40 characters** of the description for search
- Front-load a natural keyword phrase in the opening sentence
- Write for the buyer, not the algorithm — conversational and scannable
- First paragraph: what it is, who it's for, why it's special
- Include: material, width options, comfort fit details, engraving info, warranty
- End with clear ordering instructions if applicable
- No keyword stuffing — Etsy penalizes unnatural repetition
- Descriptions should NOT duplicate the title verbatim

### Algorithm Signals (2026)
- **Listing Quality Score**: clicks, favorites, purchases relative to impressions
- **Recency boost**: new/renewed listings get 24-72 hour ranking lift — optimize BEFORE publishing
- **Shop-level trust**: review count, rating, response time, shipping history all affect ranking
- **Free US shipping** listings get a dedicated ranking boost
- **Conversion rate** matters more than click volume
- Etsy's AI (Gift Mode, search) rewards clear, descriptive, natural-language listings

## Workflow

### When given a listing to optimize:

1. **Identify the product** — material, color, inlay, width, style (domed/flat/beveled), special features
2. **Generate the title** using this formula:
   ```
   [Primary Keyword Phrase] | [Secondary Keyword + Modifier] | [Occasion/Recipient/Feature]
   ```
   - Primary keyword: the most specific, high-intent search phrase for this exact ring
   - Secondary: a variation or complementary keyword cluster
   - Tertiary: occasion, recipient, or standout feature
   - Target 120-140 characters — use the space, but don't stuff

3. **Generate exactly 13 tags** following this priority framework:
   ```
   Tags 1-3:  Core product terms (what it IS) — e.g., "tungsten wedding band", "mens black ring"
   Tags 4-6:  Material/feature specifics — e.g., "wood inlay ring", "comfort fit band"
   Tags 7-9:  Occasion/intent keywords — e.g., "gift for husband", "anniversary ring", "groom gift"
   Tags 10-11: Synonyms/alternates the title missed — e.g., "promise ring men", "his wedding ring"
   Tags 12-13: Long-tail/niche phrases — e.g., "unique mens band", "engraved ring custom"
   ```
   - Verify: each tag ≤ 20 characters
   - Verify: all 13 slots filled
   - Verify: minimal word repetition across tags
   - Verify: tags do NOT simply copy title phrases

4. **Generate the description** with this structure:
   ```
   [Opening line — keyword-rich, natural, specific to this ring]

   [1-2 sentences on what makes this ring special — material, inlay, visual appeal]

   [Specs paragraph — material, width options, comfort fit, weight feel]

   [Engraving section — what's included free, how to submit custom text/fingerprint]

   [Warranty section — lifetime sizing, lifetime warranty, ships in 1-2 days]

   [Ordering instructions — size selection, how to send engraving details]

   [Care note — honest note about plating wear if applicable]
   ```

5. **Output format** — present each optimized listing in a clean, copy-paste-ready format:
   ```
   === LISTING: [Ring Name or Identifier] ===

   TITLE (X/140 chars):
   [optimized title]

   TAGS (13/13):
   1. [tag] (X chars)
   2. [tag] (X chars)
   ...
   13. [tag] (X chars)

   DESCRIPTION:
   [full description]

   === END LISTING ===
   ```

### When auditing existing listings:

1. Pull the current title, tags, and description
2. Score against these criteria:
   - Title: primary keyword front-loaded? Under 140 chars? No repetition? Readable?
   - Tags: all 13 filled? Under 20 chars each? No title duplication? Occasion tags present?
   - Description: keyword in first 40 chars? Specs included? Engraving/warranty mentioned?
3. Flag specific issues with before → after examples
4. Generate the optimized replacement

### When doing batch optimization:

- Process listings one at a time in the output
- Maintain consistent brand voice across all listings
- Ensure each listing targets a DISTINCT primary keyword — avoid keyword cannibalization
  across the catalog (e.g., don't make 5 listings all target "black tungsten wedding band")
- Call out cannibalization risks when you see them

## Common Mistakes to Flag

1. **Title keyword stuffing** — "Tungsten Ring Mens Tungsten Wedding Band Tungsten Carbide Ring" (word "tungsten" 3x)
2. **Duplicate tags** — "tungsten ring" and "tungsten band" waste a slot; combine or diversify
3. **Empty tag slots** — never leave any of the 13 unused
4. **Single-word tags** — "tungsten" alone is nearly useless; "tungsten wedding band" captures real searches
5. **Title-tag duplication** — if "black tungsten wedding band" is in the title, don't repeat it as a tag
6. **Missing occasion/gift tags** — huge traffic driver; always include 2-3 occasion/intent tags
7. **Generic filler descriptions** — "This ring is beautiful and perfect for any occasion" says nothing
8. **No engraving/warranty mention** — these are Aydins' biggest differentiators; always include them
9. **Ignoring plating honesty** — plated colors (black, gold, blue, etc.) wear over time; mention care info
10. **Keyword cannibalization** — multiple listings competing for the identical primary phrase
