## ESP Setup Notes

- Replace `{LOGO_URL}`, `{COMPANY_ADDRESS_PLACEHOLDER}`, `{COMPANY_PHONE_PLACEHOLDER}`, `{UNSUBSCRIBE_URL}`, `{PREFERENCES_URL}` in ESP.
- Append UTM tags to every PDP link: `?utm_source=email&utm_medium=email&utm_campaign=week3-revamp&utm_content=<ringname>` (COMET, ARBORIX, GARROS, MARAR, VANTA, shop-all).
- Test on iOS Mail, Gmail (web + mobile), Outlook (light + dark).
- Fonts: Georgia is a safe web-safe serif that keeps the polished/masculine tone. If you have a brand serif loaded, swap in with fallback `Georgia, 'Times New Roman', serif`.
- Alt text: when you add ring hero images above each block, use `alt="{RING NAME} — {one-line description}"`.
