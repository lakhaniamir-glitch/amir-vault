## Image Strategy — Gemini Prompts + Naming + Alt Tags

### A. Gemini (Nano Banana Pro) Image Enhancement Prompts

You're not creating new rings — you're enhancing existing product photos to make them feel premium without misleading buyers.

#### Prompt 1 — Studio enhancement (use on existing white-background shots)
```
Enhance this product photo of a men's wedding ring. Keep the ring exactly as shown — same metal, color, inlay, dimensions. Do NOT modify the ring itself in any way.

Improve only:
- Background: replace pure white with a subtle warm gradient (cream to soft beige), photographed-on-paper feel
- Add a soft natural shadow under the ring (not harsh, like window light from above-left)
- Slightly increase ring detail clarity — sharpen inlay texture, polish, brushed finish patterns
- Maintain accurate color — do not warm/cool the metal tones

Output: high-resolution square (2000x2000), product photography aesthetic, no text, no logos.
```

#### Prompt 2 — Lifestyle / on-hand shot (use to add a single hand shot per listing)
```
Generate a lifestyle photograph of this exact men's wedding ring being worn on a man's hand. Match the ring exactly — same metal, color, inlay, finish, width.

Hand details:
- Adult male hand, 30-45 years old, mild outdoor texture (not too soft, not too rough)
- Neutral skin tone, no jewelry on other fingers, clean nails
- Hand positioned naturally — fingers slightly curled, back of hand visible at a 3/4 angle
- Wedding band on ring finger of left hand

Setting:
- Soft natural daylight from a window (left side)
- Out-of-focus background — neutral neutral tones (warm beige or soft sage), no busy details
- Shallow depth of field — ring in sharp focus, background blurred

Output: 2000x2000 square, photographic realism, no text, no logos.
```

#### Prompt 3 — Detail / scale shot (use to add one zoomed detail shot per listing)
```
Generate an extreme close-up macro photograph of this exact men's wedding ring. The image should focus on showcasing the [INLAY/FINISH/PATTERN] detail.

Match the ring exactly — same metal, color, inlay material, dimensions.

Composition:
- Top-down angle showing the band laying on a textured neutral surface (slate, raw linen, or warm wood grain)
- Macro focus on the inlay/pattern — every grain, every fleck visible
- Soft natural lighting from one side, gentle shadow on the opposite
- Ring fills 60% of frame, background out of focus

Output: 2000x2000 square, ultra-sharp, photographic realism, no text or logos.
```

> **Hard rule:** Do NOT use Gemini-generated images that change the ring's appearance. Compare every output to the actual product before posting. If Gemini drifts on color, inlay pattern, or width — discard.

---

### B. Image Naming Convention

Format: `[sku-lowercase]-[shot-type]-[number].jpg`

**Shot types:**
- `studio` — enhanced white/gradient background product shot
- `hand` — on-hand lifestyle
- `detail` — macro close-up
- `pair` — couples set both rings
- `box` — ring in presentation box (if you have one)
- `scale` — ring next to coin/finger for size reference

**Examples:**
```
osias-studio-01.jpg
osias-hand-02.jpg
osias-detail-03.jpg
kilauea-studio-01.jpg
kilauea-hand-02.jpg
kilauea-detail-03.jpg
ayssagp-studio-01.jpg
ayssagp-detail-02.jpg
```

**Why this matters for SEO:** Etsy reads the filename as a soft signal. `osias-studio-01.jpg` tells Etsy nothing. `lapis-tungsten-mens-wedding-ring.jpg` tells Etsy what's in the image. **Better convention:**

```
[primary-keyword]-[sku]-[shot-type].jpg
```

Examples:
```
lapis-tungsten-ring-osias-studio.jpg
lapis-tungsten-ring-osias-hand.jpg
lapis-tungsten-ring-osias-detail.jpg
lava-rock-wedding-band-kilauea-studio.jpg
celtic-dragon-ring-jdtr367-studio.jpg
fingerprint-dog-tag-ayssagp-studio.jpg
```

Use this for all batch 2 photos.
