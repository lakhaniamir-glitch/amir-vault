## Upload Workflow

1. Generate in Nano Banana Pro → download all 27 as .jpg
2. Name files per the naming convention above (`aydins_hero_lifestyle.jpg`, `material_tungsten.jpg`, etc.)
3. Upload batch to **Shopify Admin → Settings → Files**
4. Copy each file's CDN URL
5. In theme editor → Customize homepage → paste CDN URLs into each section's image field
6. (Or) I can write a follow-up script that reads a `image-mapping.json` (filename → shopify CDN URL) and programmatically sets image fields in `templates/index.json`, then pushes.
