# Aydins Logo Set (Claw, 2026-05-15)

Premium repositioning. Yellow color dropped. Flat 2D serif wordmark, no 3D, no gems, no bevels. Reference brands: Tiffany, Cartier, Mejuri, AUrate, Catbird.

## Use rules

- **Primary mark** = `aydins-wordmark.svg`. Always prefer this over the lockup.
- **Color** = ink `#1A1A1A` on light backgrounds; cream `#FAF8F4` on dark backgrounds. Do not use the brass `#B08D57` as a fill color for the logo itself. Brass is an accent color in the broader V5 system (button fills, CTA underlines), never inside the wordmark.
- **Minimum size** = wordmark at 80px wide on screen, 1.0 inch wide in print. Below that, switch to the social `A` monogram.
- **Clear space** = padding equal to the height of the letter `A` on all four sides. No other graphics inside the clear space.
- **Do not** stretch, recolor outside the approved palette, add drop shadows, apply 3D bevels, embed gems or sparkles, or rotate.
- **Do not** add the yellow triangle from the prior brand. That brand is retired as of 2026-05-15.
